About This Project

What is this?

This is an interactive birthday greeting website designed to create a memorable and delightful experience for someone special. The page features animated balloons, decorative lights, a beautiful cake, and a heartfelt message.

Features

  • Interactive light bulbs that glow on command
  • Happy birthday music that plays automatically
  • Animated balloons that fly across the screen
  • Beautiful birthday cake with candles
  • Step-by-step reveal of birthday messages
  • Fully responsive design for mobile and desktop

How It Works

The experience is designed as an interactive journey. Visitors can click through various buttons to progressively reveal different elements of the birthday celebration, from turning on decorative lights to reading a personalized message.

Technology Stack

HTML5 CSS3 JavaScript jQuery Bootstrap LESS

Purpose

This project was created with love to celebrate a special day. It demonstrates how web technologies can be used creatively to bring joy and create memorable digital experiences for loved ones.
