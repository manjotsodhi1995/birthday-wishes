About This Project

A birthday greeting made with love and code

What Is This?

This is an interactive birthday greeting card built entirely with web technologies. It features animated lights, floating balloons, a birthday cake with candles, music, and a heartfelt message — all designed to make someone's special day even more memorable.

How It Works

The experience unfolds step by step through interactive buttons:

  • Turn On Lights — Colorful bulbs light up and the room comes alive
  • Play Music — A birthday song starts playing in the background
  • Let's Decorate — A festive banner drops down from above
  • Fly With Balloons — Balloons float and dance across the screen
  • Most Delicious Cake Ever — A beautiful CSS cake appears
  • Light Candle — Candle flames flicker to life
  • Happy Birthday — Balloons spell out the name
  • A Message For You — A personal birthday message unfolds

Technologies Used

HTML5 CSS3 jQuery Bootstrap 3 LESS CSS Animations
